# CMall网上商城

## 测试环境

- Manjaro Linux

- python 3.7.6

- mysql 8.0.20

## 依赖包

- flask

- flask_login

- flask_wtf

- wtforms

- werkzeug

- flask_sqlalchemy

- pymysql

## 运行测试

1. 自行修改config.py中的`SECRET_KEY`和`SQLALCHEMY_DATABASE_URI`

2. 确保有app/static/images/goods/目录用于存放上传的商品图片

3. `python run.py` 运行测试版程序
